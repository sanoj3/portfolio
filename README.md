#  Personal Portfolio – Django Web Application

##  Project Overview
This is a dynamic personal portfolio web application built using Django. It showcases projects, skills, and personal information with an interactive and responsive UI.

The application uses a database-driven approach to manage content, allowing easy updates and scalability.

---



###  Profile System
- Display personal details (name, bio, experience, location)
- Resume download functionality
- Profile image upload

###  Project Management
- Add and manage projects via database
- Multiple images per project (carousel view)
- Project descriptions, technologies, and features
- GitHub and live demo links for each project

###  UI/UX Features
- Responsive design using Bootstrap
- Image carousel for project showcase
- Modal-based project preview
- Clean and modern interface

---

##  Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **Version Control:** Git & GitHub

---

##  Demo Flow

1. Visit homepage  
2. View profile and personal details  
3. Browse projects  
4. Open project details (modal/carousel)  
5. Access GitHub or live demo links
